本题主要考查对乘法运算的理解以及数组的认识，思路如下：

首先思考正常的两数乘法，无非是取一个数的一位和另一个数的一位相乘，再根据坐在位置进行乘10判断，在加上进位和之前进行乘法操作后该位置的值并进位，但是这个题明确说了不能用大数类型的库（如BigInteger），这里也是在暗示我们如果直接相乘肯定会溢出，所以我们考虑开一个数组存储结果的每一位，再模拟乘法的真实过程，最后转字符串时去掉前面的0即可得出结果

class Solution {
    public String multiply(String num1, String num2) {
        int len1 = num1.length();
        int len2 = num2.length();
        if (len1 == 0 || len2 == 0)
            return "0";
        int[] res = new int[len1 + len2];
        for (int i = len1 - 1; i >= 0; i--) {
            for (int j = len2 - 1; j >= 0; j--) {
                int mul = (num1.charAt(i) - '0') * (num2.charAt(j) - '0');
                int lowIndex = i + j + 1;
                int highIndex = i + j;
                mul += res[lowIndex];
                res[lowIndex] = mul % 10;
                res[highIndex] += mul / 10;
            }
        }
        int index = 0;
        StringBuilder sb = new StringBuilder();
        while (index < len1 + len2 && res[index] == 0)
            index++;
        while (index < len1 + len2)
            sb.append(res[index++]);
        return sb.length() == 0 ? "0" : sb.toString();
    }
}
